# Equação Exponencial

## 1. Definição
- Uma equação exponencial é uma expressão algébrica que possui uma igualdade e pelo menos uma incógnita em um de seus expoentes.
- Para ser considerada equação, uma expressão precisa ter pelo menos uma incógnita, que é um número desconhecido representado por uma letra, e uma relação de igualdade. Dessa maneira, as equações exponenciais são aquelas que possuem pelos menos uma incógnita no expoente e bases positivas diferentes de 1.
- Representação:
  - a<sup>x</sup> = b
- Resolver uma equação é encontrar o valor numérico das incógnitas que aparecem nela.

Exemplo: 3^x = 27  

1. Para encontrar o valor do expoente a manipulação da equação deve ser feita a fim de que ambos os lados da equação possuam a mesma base.
2. Para isso decompõe-se o 27 em fatores primos para igualar a base ⟶ 27/3 = 9 ⟶ 9/3 = 3 ⟶ 3/3 = 1 logo 27 é igual a 3^3
3. Com as bases iguais pode-se elimina-las e trabalhar so com os expoentes, mantendo a igualdade ⟶ 3^x = 3^3 ⟶ x = 3
4. Resuldado: x = 3

Exemplo 2: 2<sup>x + 4</sup> = 64  

1. Para encontrar o valor do expoente a manipulação da equação deve ser feita a fim de que ambos os lados da equação possuam a mesma base.
2. Para isso decompõe-se o 64 em fatores primos para igualar a base ⟶ 64/2 = 32 ⟶ 32/2 = 16 ⟶ 16/2 = 8 ⟶ 8/2 = 4 ⟶ 4/2 = 2 ⟶ 2/2 = 1 logo 64 é igual a 2^6
3. Com as bases iguais pode-se elimina-las e trabalhar so com os expoentes, mantendo a igualdade ⟶ 2<sup>x + 4</sup> = 2^6
4. Isola-se o x ⟶ x + 4 = 6 ⟶ x = 6 - 4 ⟶ x = 2
5. Resuldado: x = 2

> [!TIP] DICA:
> - Para manipular a equação  e igualar os valores dos 2 membros é comum utilizar as propriedades da potenciação, radiciação e frações.
